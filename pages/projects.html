Meditrack App


Meditrack is a Java-based application designed to streamline patient data management in healthcare settings. Developed with a MySQL database, this app enables efficient storage and retrieval of patient information, including their name, age, symptoms, and blood pressure. Based on these inputs, Meditrack generates a prescription from the doctor, ensuring timely and accurate treatment recommendations.


Key Features:


  • Database Integration: MySQL backend for secure data storage and seamless access.
  • Patient Data Recording: Easy input for key patient details and symptoms tracking.
  • Automated Prescription: Matches patient details with pre-set doctor recommendations, simplifying the prescription process.

Tic-Tac-Toe Game

This classic Tic-Tac-Toe game is a Java-based program developed as an entry-level project to demonstrate core programming skills. Designed for two players, the game enables a quick, interactive experience by allowing users to mark positions on a 3x3 grid, aiming to align three symbols in a row, column, or diagonal. With a straightforward UI and smooth gameplay, this project showcases foundational concepts like control structures, conditional checks, and user input handling.


Key Features:


  • Simple UI: Clean and minimalistic grid layout for ease of play.
  • Gameplay Logic: Real-time win/draw checks and error handling for user moves.
  • Interactive Play: Supports two-player functionality, encouraging engagement.
